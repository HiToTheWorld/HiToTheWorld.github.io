MD Reader Simulation

This project converts an input/MD file to html in a controled environment. To test this on your own, download THIS file, and connect it to an html file with a <div class="md"></div> element. Anything you put in the div will be decoded according to markdown rules. Take the following as an example:

#Title ##Header1 ###Header2 ####Header3 ... ######Header5 Paragraph text _Italic text_ - unordered - list - _italic list item_ 1. ordered list 2. _Italic ordered list item_
